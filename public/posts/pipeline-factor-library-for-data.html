using data

For those interested in using data feeds like earnings calendar, news sentiment, and more within their algorithms, this listing provides a library of factors for each data feed. Within each of the threads below you'll find a number of different template factors along with the research behind it, in-sample and out-of-sample template algorithm performance, as well as a pyfolio analysis of the results.

This listing will be updated as new template factors are created. These factors are meant for you to build, iterate on, and use in other Pipeline based algorithms you may have.

At the bottom of this thread you'll also find a list of strategies, notebooks, and algorithms using datasets found on https://www.quantopian.com/data. If I've missed any please let me know.

Most popular algorithm examples using data

  • Crowdsourced post earning announcement drift strategy - A simple post-earnings announcement drift (PEAD) trading strategy that attempts to profit off the difference between reported earnings and earnings estimates. Unlike normal PEAD strategies, this uses crowdsourced estimates provided by Estimize.
  • Long/short earnings sentiment trading strategy - A variation of an earnings drift strategy that also uses Accern's Alphaone news sentiment to predict the direction of the drift.
  • Avoiding trades during earnings announcements - Simple framework to reduce risk and volatility by avoiding securities near an earnings announcement using EventVestor's earnings calendar.
  • VIX Term Structure Impact - This estimates the 'term structure impact' and uses the value to determine entry and exit points for trading the volatility ETFs VXX and XIV.
  • Weighted StockTwits message volume by trader mood - An algorithm that uses both StockTwits message volume and PsychSignal trader mood of the top 500 most liquid securities to determine 25 long and 25 short positions rebalanced everyday
  • CNBC, WSJ, and Forbes News Sentiment as ranking methodology - This uses Sentdex's 3 day average sentiment garnered from major news sources like CNBC as a long/short ranking methodology to determine 25 long and 25 short positions.
  • USO/GLD pairs trading algorithm with real price of gold - Quandl's Gold price is used to recreate a classic oil/gold pairs trading strategy found in Ernie Chan's book, Algorithmic Trading.
