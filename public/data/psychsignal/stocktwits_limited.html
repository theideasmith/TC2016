StockTwits Trader Mood (Basic Fields)
The mood of traders posting messages on StockTwits. Basic counts of bearish and bullish messages.
Access Status
Description
PsychSignal is the modern squawk box, ready to use for your Quantopian algorithm. This dataset tracks the mood of traders discussing stocks on StockTwits. This basic version of the StockTwits dataset provides raw counts of how many bullish and bearish messages PsychSignal has tracked on StockTwits.
How to use
# For use in your algorithms
# Using the full paid dataset in your pipeline algo
from quantopian.pipeline.data.psychsignal import stocktwits_limited

# Using the free sample in your pipeline algo
from quantopian.pipeline.data.psychsignal import stocktwits_limited_free

# For use in Quantopian Research
# Using the full paid version in research
from quantopian.interactive.data.psychsignal import stocktwits_limited

# Using the free sample in research
from quantopian.interactive.data.psychsignal import stocktwits_limited_free
Key Metrics
bull_scored_messages - total count of bullish sentiment messages scored by PsychSignal's algorithm
bear_scored_messages - total count of bearish sentiment messages scored by PsychSignal's algorithm
total_scanned_messages - number of messages coming through PsuchSignal's feeds and attributable to a symbol regardless of whether the PsychSignal sentiment engine can score them for bullish or bearish intensity
